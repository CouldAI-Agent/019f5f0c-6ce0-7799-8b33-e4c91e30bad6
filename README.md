# Vacation Leave Request App

A clean, responsive Flutter application that allows employees to manage their time off. 

## Features
* **Leave Dashboard:** View total available vacation days, used days, and recent leave requests.
* **Submit Requests:** An intuitive form to select start/end dates, specify the leave type (Vacation, Sick Leave, etc.), and provide a reason.
* **Status Tracking:** Track whether requests are Pending, Approved, or Rejected.

## Tech Stack
* Framework: Flutter
* Language: Dart
* Design: Material 3 

## Setup
1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to launch the app on your preferred device or emulator
